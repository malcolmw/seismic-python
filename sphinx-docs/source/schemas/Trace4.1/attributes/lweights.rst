**lweights** -- adaptive weights for line spectra
-------------------------------------------------

pointer to the array of adapative weights for all line tapers, ie
the nth weight for taper j is
*(weight + j * nfft + n),  c
or
weight(n,j)    FORTRAN

* **Field width:** 20
* **Format:** %20ld
